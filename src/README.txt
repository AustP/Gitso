These files are required for building on Windows.

IMPORTANT: When installing Python, check the "Install Just for Me" CheckBox.

1. Run python-2.5.4.amd64.msi
2. Run py2exe-0.6.9.win32-py2.5.exe
3. Run wxPython2.8-win32-unicode-2.8.4.0-py25.exe
4. Run nsis-2.46-setup.exe
5. Run pywin32-213.win32-py2.5.exe
